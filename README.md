## Overview
A free learning platform for practicing interview questions in:
- JavaScript fundamentals
- Data Structures & Algorithms (DSA) in JavaScript
- React concepts

## Why It Matters
This project helps learners prepare for technical interviews by combining **quizzes, scoring logic, and progress tracking**. It's designed to be small but realistic, with a senior-level architecture mindset.

## Features
- **Default Practice** — built-in curated questions for JS, DSA, and React (no setup needed)
- **AI Generate** — 10-question sets generated by Groq AI, with 3–5 questions recycled from previous sets for spaced repetition
- **Create Quiz** — build your own custom quizzes
- Instant scoring with per-question explanations
- Progress tracking dashboard (avg score, best score, weak areas)
- Firebase authentication (email/password + Google)
- Session reset on logout — clean state for every login

## Project Structure
```
src/
├── components/
│   ├── TopicSelector/    # Choose JavaScript, DSA, or React
│   ├── QuizList/         # 3-section browse: Default / AI / Create
│   ├── QuizPlayer/       # Take quizzes
│   ├── ResultPage/       # Score + explanations
│   ├── Dashboard/        # Progress + all quiz sections
│   └── QuizForm/         # Create custom quizzes
├── store/
│   ├── slices/
│   │   ├── quizSlice.js   # Quiz data + static/firestore quizzes
│   │   ├── resultSlice.js # Scores and history
│   │   └── userSlice.js   # Auth state + authReady flag
│   └── index.js
├── services/
│   ├── quizService.js    # Firestore CRUD (handles static-* IDs locally)
│   ├── authService.js    # Firebase login
│   └── aiService.js      # Groq AI generation with recycling logic
├── data/
│   ├── staticQuizzes.js  # Built-in quiz definitions (no Firestore needed)
│   └── questions/        # Raw question banks (JS, DSA, React)
├── firebase/
│   └── config.js
└── utils/
    └── scoring.js        # Scoring logic
```

## Installation

1. Clone the repository:
```bash
git clone https://github.com/tanhoang0803/interview-quiz-space.git
cd interview-quiz-space
```

2. Install dependencies:
```bash
npm install
```

3. Create `.env` from the template:
```bash
cp .env.example .env
# Fill in your Firebase and Groq keys
```

4. Run the app:
```bash
npm start
```

5. Build for production:
```bash
npm run build
```
